# Online Ticket Booking System

## Project Description
The Online Ticket Booking System is a Java-based application that allows users to book tickets online. 
The system provides functionalities such as ticket booking, viewing available tickets, and managing reservations.

## Technologies Used
- Java
- OOP Concepts
- File Handling
- Database 

## Features
- User can book tickets
- View available seats
- Cancel tickets
- Simple console-based interface

## How to Run
1. Download the project
2. Compile using: javac Main.java
3. Run using: java Main
